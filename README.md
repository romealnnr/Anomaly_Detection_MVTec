## 📦 Download the MVTec AD dataset

The MVTec AD dataset is not included in this repository (1.2 GB).

Please download it from the official page:
https://www.mvtec.com/company/research/datasets/mvtec-ad

1. Fill out the form
2. Download the `mvtec_anomaly_detection.tar.xz` file
3. Place it in the project folder
4. Extract:
tar -xf mvtec_anomaly_detection.tar.xz
5. Rename:
mv mvtec_anomaly_detection mvtec
